Logging Module¶

This module contains functionality for logging. It mainly configures the log level and a handler so that logging can coexist with the tqdm progress bar.

class stransfer.c_logging.TqdmLoggingHandler(stream=None)[source]¶

Custom logging handler. This ensures that TQDM progress bars always stay at the bottom of the terminal instead of being printed as normal messages

emit(record)[source]¶

Emit a record.

If a formatter is specified, it is used to format the record. The record is then written to the stream with a trailing newline. If exception information is present, it is formatted using traceback.print_exception and appended to the stream. If the stream has an ‘encoding’ attribute, it is used to determine how to do the output to the stream.
